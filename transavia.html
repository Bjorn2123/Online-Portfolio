Transavia reisverhalen

Transavia Reisverhalen is een full responsive website van Transavia waarmee de gebruiker voor, tijdens of na zijn reis verhalen kan lezen. Als voorbereiding op de reis kan de gebruiker verhalen toevoegen aan zijn readinglist, zodat die verhalen offline te lezen zijn. Wanneer de gebruiker tijdens zijn reis een verhaal naar wens leest kan hij dit verhaal toevoegen aan zijn favorieten om het zo nogmaals te kunnen lezen.

De full responsive website is volledig met HTML5, CSS en Javascript geprogrammeerd en is toegankelijk op iedere device. Omdat de doelgroep van Transavia altijd onderweg is ligt de focus op Mobile first om zo aan de context van de gebruiker te kunnen voldoen.
